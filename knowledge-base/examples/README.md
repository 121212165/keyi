# Example Entries

This directory contains example entries for different difficulty levels and issue types.

## Purpose

These examples serve as references for creating new knowledge base entries. They demonstrate proper formatting, structure, and level of detail required.

## Examples

- [Low Difficulty Example](low_difficulty_example.md) - Simple issue with straightforward solution
- [Medium Difficulty Example](medium_difficulty_example.md) - Moderate complexity issue
- [High Difficulty Example](high_difficulty_example.md) - Complex issue requiring comprehensive documentation

## Usage

When creating a new entry:
1. Review the appropriate example for your difficulty level
2. Copy the relevant template from [docs/ISSUE_TEMPLATE.md](../../docs/ISSUE_TEMPLATE.md)
3. Fill out all required sections
4. Follow the structure and detail level shown in examples
