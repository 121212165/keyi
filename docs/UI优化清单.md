## 优化方向

### 配色方案建议

```css
/* 温暖、专业的心理应用配色 */
--color-primary: #5B7FA6;  /* 柔和蓝（专业、信任） */
--color-warm: #E8B89D;      /* 暖杏色（温暖、关怀） */
--color-accent: #A8C8A8;    /* 柔和绿（平静、放松） */
--color-background: #FAF8F5; /* 米白（柔和、舒适） */
```

### 设计风格建议

1. **圆润造型**：圆角、气泡、按钮使用柔和圆角
2. **渐变色彩**：背景使用柔和渐变
3. **情感化**：欢迎语、提示语更温暖
4. **简洁清晰**：信息层级分明
