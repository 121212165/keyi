# 每日心理汇总功能 - 灵感文档

> 创建日期：2026-02-14

---

## 一、需求来源

用户希望将每天的历史对话进行心理化总结利用，作为后续分析和洞察的基础。

---

## 二、功能设想

### 核心能力
- [x] 每日自动汇总对话
- [x] 情绪趋势分析（曲线图）
- [ ] 主题/关键词提取（待定）
- [ ] 心理状态评估（待定）

### 输出形式
- [x] 文本报告
- [x] 情绪曲线图（展示7天趋势）
- [ ] 可导出文件（待定）

---

## 三、功能规格

| 功能 | 规格 |
|------|------|
| 触发方式 | 每天自动汇总（定时任务） |
| 内容深度 | 详细的心理分析报告 |
| 数据范围 | 展示最近7天趋势 |
| 存储 | 永久保存 |
| 权限 | 用户拥有第一裁定权（可查看/导出/删除） |

## 五、设计灵感

### 报告内容结构
```
1. 今日概览
   - 对话数量
   - 主要话题

2. 情绪分析
   - 情绪分布
   - 情绪变化曲线（7天趋势图）

3. 主题归纳（待定）
   - 讨论了哪些话题
   - 关注点在哪里

4. 洞察与建议
   - AI观察到的模式
   - 小贴士/建议

5. 风险标记（如有）
   - 需要关注的信号
   - 资源推荐
```

### 用户权限
- [x] 查看自己的报告
- [x] 导出报告
- [x] 删除报告
